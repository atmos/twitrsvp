twitrsvp
========
Another [oauth][oauth] experiment.  Organize folks with [sinatra][sinatra] and [twitter][twitter].  Try it out on [TwitRSVP][twitrsvp].

Installation
============
It's a sinatra app.

    % bundle install
    % bundle exec rake


testing
=======

Just run rake...
